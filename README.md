# Weather WebApp With Node js
Una aplicacion web para obtener el pronostico en tiempo real a traves de OpenWeatherMap API y MapBox API.

Fue desarrollada utilizando

* Node js
* Express
* Request
* Javascript
* HTML
* Handlebars
* CSS
* Git
* Heroku
